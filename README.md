# PROJECT OVERVIEW
This project is an interactive sales analytics dashboard developed using Microsoft Power BI to analyze e-commerce business performance. The dashboard provides insights into key business metrics such as total orders, average sales, and overall profit, helping identify sales trends and customer purchasing patterns.

The dashboard includes:

Sales performance overview with KPI cards (Total Orders, Average Sales, Total Profit)
Analysis of sales across different payment modes
Geographic insights by city and state
Time-based trend analysis using order date (month/quarter)
Interactive visualizations for better business decision-making
